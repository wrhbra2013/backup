Como Escrever Fluxogramas

1. Direcao do Fluxo

Defina a direcao dos nos logo apos o tipo do grafo:

SintaxeEfeito
graph TDCima para Baixo (Top-Down)
graph BTBaixo para Cima (Bottom-Top)
graph LREsquerda para Direita (Left-Right)
graph RLDireita para Esquerda (Right-Left)

2. Formas dos Nos

SintaxeFormatoUso
A[texto]RetanguloProcesso / Acao
A(texto)RoundedInicio / Fim
A{texto}RomboDecisao (Sim/Nao)
A((texto))CirculoConexao / Evento
A[(texto)]CilindroBanco de Dados
A[/texto/]ParalelogramoEntrada/Saida
A[/texto\\]TrapezioManual

3. Conexoes

SintaxeEfeito
A --> BSeta simples
A --- BLinha sem seta
A -.-> BSeta pontilhada
A ==> BSeta grossa
A -->|texto| BSeta com rotulo
A -->|texto| BSeta com texto

4. Subgrafos (Grupos)

subgraph Nome do Grupo A --> B end

5. Exemplo Completo

graph TD // Definir nos A[Inicio] --> B{Condicao?} B -->|Sim| C(Processar) B -->|Nao| D[Sair] C --> E((Fim)) // Estilo dos nos style A fill:#4CAF50,color:#fff style E fill:#f44336,color:#fff

6. Estilos e Cores

SintaxeEfeito
style N fill:#corMuda cor de fundo do no
style N stroke:#corMuda cor da borda
style N color:#corMuda cor do texto
style N stroke-width:2pxEspessura da borda
classDef nome fill:#corDefine uma classe de estilo
class N nomeAplica classe a um no

7. Dicas Uteis

  • Identificadores dos nos sao unicos e case-sensitive: A e a sao diferentes
  • Use espacamento consistente para manter o codigo legivel
  • O editor suporta Tab para indentacao
  • Voce pode copiar/colar exemplos do editor diretamente
  • Exporte como PNG ou SVG depois de renderizar
