Como configurar o Instagram Graph API

1. Criar App no Facebook Developers

  1. Acesse developers.facebook.com
  2. Clique em "Meus Apps" → "Criar App"
  3. Escolha o tipo "Business"
  4. Dê um nome ao app (ex: "Meu Feed IG")
  5. Clique em "Criar App"

2. Adicionar o producto Instagram Graph API

  1. No painel do app, clique em "Configurar" ao lado do Instagram Graph API
  2. Ou vá em "Produtos" → busque "Instagram" → clique em "Configurar"
  3. Conecte uma conta Instagram Business/Creator

3. Converter conta para Business/Creator (se necessario)

  1. No app Instagram, vá em Configuracoes → Conta
  2. Toque em "Mudar para Conta Profissional"
  3. Escolha "Criador" ou "Negocio"
  4. Siga as instrucoes e vincule a uma Pagina Facebook

4. Vincular conta IG a uma Pagina Facebook

  1. No Facebook, crie ou acesse uma Pagina
  2. Vá nas Configuracoes da Pagina → Contas Profissionais
  3. Clique em "Conectar conta" e selecione sua conta Instagram
  4. Autorize o vinculo

5. Configurar permissoes do App

  1. No painel do app, vá em "Configuracoes" → "Permissoes da API"
  2. Ative as permissoes:
    • instagram_basic
    • pages_show_list
    • pages_read_engagement
  3. Para buscar paginas publicas tambem ative:
    • Page Public Content Access (em "Recursos")

6. Gerar Access Token

  1. Vá ao Graph API Explorer
  2. No canto superior direito, selecione seu app
  3. Clique em "Gerar Token de Acesso"
  4. Selecione as permissoes: instagram_basic, pages_show_list, pages_read_engagement
  5. Clique em "Gerar Token de Acesso"
  6. Copie o token gerado

7. Converter para Token de Longo Prazo (recomendado)

  1. No Graph API Explorer, copie o token curto
  2. Cole no campo acima e clique em "Buscar Feed"
  3. O token curto expira em ~1h. Para tokens longos:
  4. Use o endpoint: GET /oauth/access_token?grant_type=fb_exchange_token&client_id=APP_ID&client_secret=APP_SECRET&fb_exchange_token=SHORT_TOKEN

Erros comuns:

  • "Object does not exist" → O ID nao e valido ou a conta nao e Business
  • "Missing permission" → Falta ativar a permissao no app
  • "Unsupported get request" → Token invalido ou expirado
  • "pages_read_engagement" → Ative Page Public Content Access
